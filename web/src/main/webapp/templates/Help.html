Overview

The Jive Wifi Management tool allows admins to manage wireless networks and individual wireless access points. Admins can manage bandwidth, users, and devices. Admins can also limit bandwidth, block users and devices from using the wireless network.

Global Tips

The pages that have a search box in the upper right corner of the page, allow for admins to search for items on that page. The pages that have a search bar are:

  • Access Points
    • Searchable by:
      • MAC address
      • IP address
  • Wireless Networks
    • Searchable by:
      • Name
  • Devices
    • Searchable by:
      • MAC address
      • IP address
  • Sites
    • Searchable by:
      • Name
      • City
      • Account ID

With any change, the admin must click the button for the changes to persist.

Only Ubiquiti Unifi Access Points with supported firmware versions will work with this tool.

We do not support Internet Explorer 8. Please download Google Chrome, Firefox or update Internet Explorer to 9 or 10.

Any questions please contact us at customersupport@getjive.com

Access Points

This section helps admins adopt, remove and manage individual access points(AP’s). This screen is where AP’s are adopted, removed, and sorted. Each line has a MAC address which links to the page to edit the AP. Admins can also click on the gear icon at the end of the line to see the options available to configure the IP address of the AP. Clicking on “Settings” allows users to change the ‘Alias’ or name of the device that shows up in the Wifi Management tool.

If Static IP address is chosen, an extra “Network” tab shows up next to the tab “Config”. In the Network tab, there are fields to configure the Static IP address.

Wireless Networks

This section helps admins manage the wireless networks that can be shared on many access points across a site. They can set up several networks with different rules, and security options.

To add a network, click on the button. In this new window admins have can change the network name, set the status, security password and enable/disable a guest policy. When guest policies are enabled the settings from the “Site Settings” are enforced. Advanced options include setting up a VLAN, broadcasting the SSID, and options for the type of security and encryption being used.

Connected Devices

This section is where end user devices are listed. The gear icon on the right of each line has two options. Admins can block/unblock users and authorize/unauthorize users. Admins can also sort, block and limit devices.

Click on the MAC address (dc:9b:9c:14:b2:ff) link, to show an overview of network activity for the end user device. From this new screen admins can also block/unblock users and authorize/unauthorize users.

Clicking on the button, shows options to include extra columns, such as:

  • Authorization
  • Blocked
  • Device

List of column headers and what they describe:

  • Network
    • Which network the device is connected to
  • Status
    • Connected/Disconnected
  • Authorization
    • User
    • Unauthorized
    • UserGuest
  • Download
    • How much data has been downloaded
  • Upload
    • How much data has been uploaded

Sites

This section allows the admin to add and sort sites. To add a site click on the button. The new screen will give the user options to enter:

  • Site name
  • Email for alerts
  • Address
  • Assigned Devices
    • Enter the MAC address of each Access Point you want associated with this site.

FAQ

My AP is plugged in and is set to inform http://wifi.jive.com:8080/inform and it is not showing up for me to adopt the AP, what is wrong?

  • The wrong site is selected
    • In the top left corner there is a drop down to select different sites
  • You did not add the MAC address of the AP to the correct site
    • Go to “Sites”
    • Select the Site you want the AP to connect to
    • Click on “Assigned Devices”
    • Add the MAC address of the access point
    • Click “Update” to save the changes

My AP is plugged in but it is not showing up in my Site to adopt. There are 2 options:

  • SSH into the device and copy/paste this command:
    • ssh ubnt@<ip-address-of-access-point> (e.g. ssh ubnt@192.168.10.4)
    • Password is: ubnt
    • Then type the following once you have made a connection:
      • set-inform http://wifi.jive.com:8080/inform
OR
  • Open up the “Unifi Discovery Utility” link to utility tool
    • Wait for the tool to find the access point
    • Click on “Manage”
    • On the next window select “Set Inform”
    • Type in: http://wifi.jive.com:8080/inform
    • Select “Apply”
Go back to the browser, and browse to wifi.jive.com and click on “Access Points”. Find the MAC address that you want to adopt. Click the gear Icon on the right and select “Adopt”. After a few minutes the Access Point should say “Connected” under “Status”
